# Campaign Assets

Placeholders para imágenes de campañas promocionales.

## Requisitos:
- Formato: PNG
- Dimensiones recomendadas: 800x400 px
- Tamaño máximo: 300 KB por imagen
- Compresión: TinyPNG, ImageOptim o similar

## Imágenes requeridas:
1. campaign_insurance.png - Campaña de seguros/SOAT
2. campaign_catalog.png - Campaña de catálogo de repuestos
3. campaign_maintenance.png - Campaña de mantenimientos pendientes

## Producción:
Reemplazar estos placeholders con assets reales optimizados.
